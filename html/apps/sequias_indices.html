A través de la siguiente aplicación es posible calcular el Standardized Precipitation Index (SPI), el Standardized Precipitation Evapotranspiration Index (SPEI), el Vegetation Condition Index (VCI), el Temperature Condition Index (TCI), el Soil Vegetation Index (SVI) y el Vegetation Health Index (VHI), seleccionando las fechas de referencia y la zona de interés.

¿Cómo se calculan y qué representan estos índices?

Standardized Precipitation Index (SPI)

El Standardized Precipitation Index (SPI) es un índice de aridez basado en la precipitación. Mide la anomalía de la precipitación en relación con una distribución de referencia, calculando la desviación estándar de las precipitaciones en un período de tiempo determinado. Se utiliza sobretodo para evaluar las condiciones de sequía o exceso de precipitación. Su ecuación es:

SPI = P - P ( t ) σ

Los valores del SPI oscilan entre -3 y +3, donde los valores más altos indican condiciones de exceso de precipitación o mayor humedad. Un SPI cerca de 0 indica condiciones climáticas cercanas a la media histórica de precipitación, mientras valores negativos indican condiciones de sequía. Cuanto más negativo sea el valor, más severa será la sequía.

Standardized Precipitation Evapotranspiration Index (SPEI)

El Standardized Precipitation Evapotranspiration Index (SPEI) es un índice de aridez que tiene en cuenta tanto la precipitación como la evapotranspiración. Al igual que el SPI, el SPEI mide las anomalías en relación con una distribución de referencia, pero también considera las necesidades de agua de las plantas, a través de la evapotranspiración. Su ecuación es:

SPEI = P - P ( t ) - E - E ( t ) σ

El SPEI tiene un comportamiento muy similar al SPI. Sus valores varían entre -3 y +3, con valores negativos que indican condiciones secas y valores positivos que indican una mayor presencia de agua.

Vegetation Condition Index (VCI)

El Vegetation Condition Index (VCI) se basa en la diferencia entre el índice de vegetación observado (por ejemplo, NDVI o EVI) y la media histórica, normalizando el resultado dentro de un rango predefinido. El VCI se utiliza con frecuencia para monitorear el estrés de las plantas y la disponibilidad de agua en el suelo. Su ecuación es:

VCI = NDVI - NDVI ( t ) NDVI - NDVI ( min )

El VCI se normaliza generalmente en un rango de 0 a 100, con valores bajos que indican estrés hídrico y valores altos que indican una condición saludable de la vegetación o en general buenas condiciones de humedad en el suelo.

Temperature Condition Index (TCI)

Temperature Condition Index (TCI) es un índice que mide las condiciones térmicas. Puede basarse en la temperatura media, máxima o mínima, y evalúa las anomalías en relación con la media histórica. Se puede utilizar para monitorear los efectos de las temperaturas extremas o el estrés térmico en los cultivos. Su ecuación es:

TCI = T - T ( t ) T - T ( min )

El TCI no tiene valores límites específicos, ya que depende de la referencia y de el cálculo utilizado. Generalmente, valores altos indican condiciones de calor extremo o estrés térmico para las plantas, mientras que valores bajos indican condiciones de temperatura moderada y favorables para el crecimiento de las plantas.

Soil Vegetation Index (SVI)

El Soil Vegetation Index (SVI)es un índice que proporciona información sobre la salud y las condiciones de la vegetación, en relación con la disponibilidad de agua en el suelo. Se calcula utilizando índices de vegetación y datos de humedad del suelo, según la siguiente ecuación:

SVI = NDVI - NDVI ( t ) NDVI - NDVI ( min )

Este índice se normaliza generalmente en un rango de 0 a 1 o de 0 a 100. Su interpretación es opuesta a la del TCI, con valores bajos que indican estrés hídrico y valores altos que indican buenas condiciones de humedad del suelo y vegetación.
